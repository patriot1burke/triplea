About:

Triplea is a networked open source gaming engine and clone of a popular board game. Triplea is released under the Gnu Public Liscenc (gpl).

Installation

Playing:

On windows to play the game you simply double click on the triplea.bat file in the bin directory. On Mac and Linux you will have to modify the triplea.bat file to orient the slashes the correct way.
There is currently a bug that makes the game look really bad in Linux. I plan on fixing this after I get a linux box set up. If your a java developer with access to a linux box Im pretty sure the problem is in the games.strategy.triplea.image.MapImage class. I havent heard any reports of the game working or not working on macs, so Im assuming everything is fine.

Common Problems:

If you run the game and you get an error message that says 'Bad command or file name', then you havent installed or set up the java virtual machine correctly. Please see the installation page for instructions on setting up a java virtual machine on your computer.
