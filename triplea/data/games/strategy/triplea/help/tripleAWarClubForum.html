This poster is build for PBEM games via TripleAWarClub.org. Instructions:

  1. Create a new Forum post in the Play By Forum section http://www.tripleawarclub.org/modules/newbb/viewforum.php?forum=20
  2. Copy the topic id from the URL displayed in the address bar of you browser. If the URL is http://www.tripleawarclub.org/modules/newbb/viewtopic.php?topic_id=609&forum=20.0, put this number (609) into the Topic ID field in TripleA
  3. Put your login and password for TripleAWarClub.org forum into the login and password fields
  4. Click the Test Post button, to check that TripleA can reply to your forum post

* Make sure you have registered for receiving dice emails if your dice server sends them out. Click the 'Help?' button by your dice server to see if this is required.
