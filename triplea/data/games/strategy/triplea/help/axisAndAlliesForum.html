This poster is build for PBEM games via AxisAndAllies.org. Instructions:

  1. Create a new Forum post in the Play Boardgames section http://www.axisandallies.org/forums/index.php?board=40.0
  2. Copy the topic id from the URL displayed in the address bar of you browser. If the URL is http://www.axisandallies.org/forums/index.php?topic=25252.0, put this 5 digit number (25252) into the Topic ID field in TripleA
  3. Put your login and password for the AxisAndAllies.org forum into the login and password fields
  4. Click the Test Post button, to check that TripleA can reply to your forum post

* Make sure you have registered for receiving dice emails if your dice server sends them out. Click the 'Help?' button by your dice server to see if this is required.
