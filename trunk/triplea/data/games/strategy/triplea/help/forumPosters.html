Forum Poster

A forum poster can post the turn summary and save game to a forum at the end of each players turn. This allows two or more players to play a game where you don't have to be online at the same time.

The poster for each forum may be different so click help once you have selected a specific forum poster, to learn more.

* Make sure you have registered for receiving dice emails if your dice server sends them out. Click the 'Help?' button by your dice server to see if this is required.
